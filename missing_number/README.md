# Missing Integer 

## Directions
Write a function `missing_integer` that takes an unsorted list of non-negative integers and returns the missing number in the consecutive sequence. If the list passed to `missing_integer` has no missing integers or more than one missing integer, the function returns `None`.

```python
missing_integer([1, 2, 4]) == 3
missing_integer([8, 6, 9, 10]) == 7
missing_integer([1001, 998, 999, 1003, 1002]) == 1000
missing_integer([5, 6, 7, 8, 9]) == None
missing_integer([2, 4, 6, 8]) == None
```
